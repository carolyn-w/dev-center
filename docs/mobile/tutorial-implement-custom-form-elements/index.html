Implement a Custom Form Element

Geocortex Mobile comes with bundled with Geocortex Workflow. Geocortex Workflow includes a large suite of form elements to help you create custom forms to solve your business case. However, if Geocortex Workflow is missing a form element you need, you can implement a custom form element with the Geocortex Workflow SDK.
